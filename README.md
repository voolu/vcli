# Voolu CLI

Simplify terminal access to remote machines using the [voolu network](https://dev.voolu.io).

## Create an account

```bash
vacc create
```

```shell script
[?] Choose a username: username
[?] Choose a password: ********
[?] Re-enter password: ********
```

## Login

```bash
vacc login
```

```bash
[?] Username: username
[?] Password: ********

Logged in and saved token at ~/.voolu/token
```

## Connect host

```bash
vhost my-host
```

## Connect clients

```bash
vsh my-host
```
